# 广雅生存手册（GY-Guide）

所有人都可以参与编辑，内容经过审核求证之后才会加入。

# 如何贡献内容（How To Contribute）

想要贡献内容，有两种方法。第一是提交一个 issue 到这个仓库，里面写明想要增加什么内容或者有什么内容需要修改。第二是 fork 仓库到你的账号，进行修改，然后推送到这里。

### 方法一：提交 issue

写明要增加什么内容、修改什么内容。比如我想要增加一个「广雅火星校区」，那我就在 issue 里填写标题「增加章节“广雅火星校区”」，然后填写内容巴拉巴拉什么的。

### 方法二：自己动手

注：本方法适合高级用户

看到仓库页面右上角的 fork 了吗？点它，fork 到你自己账号，然后（**重要**）新建一个分支，在这个分支里面尽情修改。改完进行推送。推送一次可能不够，仓库管理者可能会写一些评论给你，让你进行修改。改好了就会被合并到主分支。
